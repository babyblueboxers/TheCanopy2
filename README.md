# The Canopy 2
Repo for The Canopy 2, a metaroom for Creatures Docking Station.

# Installation
Move "The Canopy 2.agents" file to your Docking Station Agents directory.
Delete any old Catalogue file from your Catalogue folder when upgrading.

# Notes
* This is not meant to be any kind of final, stable release. This is just for letting people play with the room as it is developed.
* There are no lifts or means to enter or exit the room. Recommend using elevines, teleporters, and other methods of transportation.
* Expect Bugs, errors, and other issues.
* No support provided.
* Feel free to submit bug reports but don't expect them to get fixed in any timely fashion.
